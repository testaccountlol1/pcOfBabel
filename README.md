# What this is

this is a project with fuse made after i saw inkbox's usb of babel and said "why not do this with a pc?", and then i made this

# Dependencies

you need gcc and fuse.h to run this and reset.sh is only for linux, if you want to run this on windows you must either translate it or if anyone else has already done that copy it over

# Reminders

for the second or later things you must use reset.sh and after that you must unmount it with fuse
